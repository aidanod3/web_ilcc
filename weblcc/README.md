# WebLCC Frontend

Single React app that serves two routes:
- `/main` (WebLCC)
- `/autograder` (Autograder)

## Local Development

Start the backend:

```bash
cd /Users/cedsolitep/web_ilcc/backend/weblcc-backend
npm install
npm run dev
```

Start the frontend:

```bash
cd /Users/cedsolitep/web_ilcc/weblcc
npm install
npm run dev
```

Vite will print the local dev URL (usually `http://localhost:5173`).
